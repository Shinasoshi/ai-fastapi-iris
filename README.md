# Iris ML API (FastAPI + scikit-learn)

## Opis
Projekt udostępnia wcześniej wytrenowany model uczenia maszynowego jako usługę webową (API).
Model rozwiązuje problem klasyfikacji zbioru Iris: na podstawie 4 cech kwiatu przewiduje gatunek.

Technologie:
- Python
- scikit-learn (model)
- FastAPI (API)
- uv (zarządzanie środowiskiem)
- Git (współpraca i historia zmian)

---

## Wymagania
- Python >= 3.11
- zainstalowane narzędzie `uv`
